# Transformer升级之路：二维位置的旋转式位置编码
**作者:** 苏剑林 (科学空间)
**日期:** 2021-05-10
**原文链接:** https://kexue.fm/archives/8397

在之前的文章中我们提出了旋转式位置编码RoPE。但是最近一段时间，Transformer模型在视觉领域也大火，于是就有了问题：二维情形的RoPE应该是怎样的呢？

## 二维位置
在CV中，图像的位置信息是二维的 $(x, y)$。如果简单展平后当作一维处理，位置 $(x+1, y)$ 和 $(x, y+1)$ 展平后的距离分别是 $h$ 和 $1$，这显然不合理。因此我们需要专门为二维情形设计位置编码。

### 标准答案
二维RoPE的一个解为分块矩阵：
$$ R_{x,y} = \begin{pmatrix} \cos x	heta & -\sin x	heta & 0 & 0 \ \sin x	heta & \cos x	heta & 0 & 0 \ 0 & 0 & \cos y	heta & -\sin y	heta \ 0 & 0 & \sin y	heta & \cos y	heta \end{pmatrix} $$
实现上就是将输入向量分为两半，一半施加 $x$ 的一维RoPE，一半施加 $y$ 的一维RoPE。

## 推导思路
二维RoPE的基本要求是满足“相对性”：$R_{x_1,y_1}^	op R_{x_2,y_2} = R_{x_2-x_1, y_2-y_1}$。同时需要“可逆性”，保证能从矩阵中无损重构位置信号。

### 途径一：四元数 (夭折)
四元数是复数的推广。但在四元数中，一般有 $e^{p+q} 
eq e^p e^q$。由于指数乘法无法转换为加法，相对性没法得到保证，此路不通。

### 途径二：矩阵指数 (通解)
矩阵指数定义为：$\exp B = \sum_{k=0}^\infty \frac{B^k}{k!}$。
如果 $AB=BA$，则 $(\exp A)(\exp B) = \exp(A+B)$。
一维RoPE可以表示为：
$$ R_n = \exp \left\{ n	heta \begin{pmatrix} 0 & -1 \ 1 & 0 \end{pmatrix} ight\} $$
对于二维RoPE，考虑 $R_{x,y} = \exp(x B_1 + y B_2)$。推导得出约束条件：
1. $B_1^	op + B_1 = 0, B_2^	op + B_2 = 0$ (反对称矩阵)
2. $B_1 B_2 = B_2 B_1$ (可交换)

对于 $4 	imes 4$ 矩阵，满足条件的通解即为上述分块旋转矩阵。

## 文章小结
本文尝试了四元数和矩阵指数两种推导过程，最终通过矩阵指数给出了二维RoPE的解，深化了对RoPE的理解。
